# Estación Nébula-7 — Juego de Thiago

Juego de detective espacial en español, hecho en HTML/CSS/JS puro (sin frameworks, sin backend, sin API keys).

## Temporada 1: "El Enigma del Cometa Fantasma"

- ✅ Caso 1: El Robo de la Piedra Lunar
- ⏳ Caso 2: El Robot Silencioso
- ⏳ Caso 3: El Cristal Robado del Bazar Estelar
- ⏳ Caso 4: La Señal Extraña
- ⏳ Caso 5: El Tripulante que No Existe
- ⏳ Caso 6 (Final): La revelación

## Cómo correrlo localmente

No necesita instalación. Con VS Code:
1. Instala la extensión **Live Server** (de Ritwick Dey)
2. Click derecho sobre `index.html` → **Open with Live Server**
3. Se abre solo en el navegador con recarga automática al guardar cambios

## Despliegue

Este repo está conectado a Netlify. Cada `git push` a la rama `main` publica automáticamente en el sitio.

## Guardado de progreso

El progreso del jugador se guarda en `localStorage` del navegador (por dispositivo). Ver el proyecto en Universe para el roadmap de una futura migración a Supabase (multi-dispositivo, multi-jugador).
